Texto Marcado

Podemos criar também um texto marcado usando a tag MARK <mark> </mark>

Podemos fazer com que todos os marks tenham uma mesma cor (menos os especificados diretamente ↑), diferente da padrão (amarelo) criando um style no head: texto exemplo 1 texto exemplo 2

Texto Grande e Pequeno

Podemos criar um texto grande e um texto pequeno usando as tagas BIG <big> </big> e SMALL <small> </small> respectivamente

Texto Deletado

Podemos marcar um texto como excluído, para dizer que não vale mais, usando a tag DEL <del> </del>

Texto Inserido/Sublinhado

Podemos marcar um texto como sublinhado, para dar ênfase de que foi inserido depois, usando a tag INS <ins> </ins> COM SEMANTICA

Podemos marcar um texto como sublinhado, para dar ênfase de que foi inserido depois, usando a tag U <u> </u> SEM SEMANTICA

Texto Sobrescrito

Podemos fazer um texto Sobrescrito usando a tag SUP <sup> </sup>

Texto Subescrito

Podemos fazer um texto Subescrito usando a tag SUB <sub> </sub>
